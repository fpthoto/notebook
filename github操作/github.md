# GitHub操作

1、查看git配置信息

git config --list



2、查看git用户名

git config user.name



3、查看邮箱配置

git config user.email



4、全局配置用户名

git config --global user.name "ftpthoto"



5、全局配置邮箱

git config --global user.email "853683088@qq.com"
